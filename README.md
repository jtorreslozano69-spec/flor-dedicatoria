# 🌻 Flor Interactiva Animada

Un proyecto web creativo que combina animaciones visuales con mensajes personalizados y música de fondo.

(se que tiene una flor pero en celular se ve mejor)
(puede ser que lo doble)

* **Navegación Dinámica:** Incluye una transición fluida entre la página de inicio y la dedicatoria principal (`index(2).html`).
* **Animaciones CSS:** Efectos visuales en los pétalos y el tallo de la flor.
* **Experiencia Sonora:** Incorpora una melodía (`cancion.mp3`) que acompaña la visualización.
* **Despliegue:** Disponible en vivo gracias a **GitHub Pages**.

* HTML5
* CSS3
* JavaScript
